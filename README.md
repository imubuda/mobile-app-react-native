# Mobile App using React Native

## Description

A comprehensive mobile application built using React Native, providing a seamless user experience across both iOS and Android platforms. This project showcases the capabilities of React Native in developing cross-platform mobile applications with ease.

### Key Features

*   **User Authentication**: Secure login and registration features to authenticate users
*   **Home Screen**: A user-friendly home screen displaying essential application information
*   **Detail Screen**: Displaying detailed information about each item, with the ability to save and navigate through items
*   **Item Management**: CRUD operations for items, including creating, reading, updating, and deleting
*   **Offline Support**: Offline capabilities to ensure seamless usage even without internet connectivity
*   **Theme Switching**: Switch between light and dark modes to enhance user preference

## Technologies Used

*   **React Native**: A framework for building native mobile apps using JavaScript and React
*   **Expo**: A set of tools and services for building and publishing React Native applications
*   **Redux**: State management library for managing global state across the application
*   **Redux Persist**: Library for persisting Redux state in the local storage
*   **React Navigation**: Navigation library for managing navigation between screens
*   **React Native Vector Icons**: Library for adding icons to the application

## Installation

### Prerequisites

*   Install Node.js (14.17.0 or higher) and npm (6.14.13 or higher) on your machine
*   Install Expo CLI by running `npm install -g expo-cli` in your terminal

### Setup

1.  Clone the repository using `git clone https://github.com/username/mobile-app-react-native.git`
2.  Navigate into the project directory
3.  Install dependencies by running `npm install`
4.  Start the application using `npm start`
5.  Open the application in an emulator or on a physical device using Expo Go app
